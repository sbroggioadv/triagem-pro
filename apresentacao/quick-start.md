# Triagem Pro — Quick Start

**Instalar e começar a usar em 15 minutos.** Sem código, sem terminal, sem stress.

---

## Pré-requisitos

1. Conta no **Claude Cowork** (claude.ai)
2. **WhatsApp Business** funcionando
3. **15 minutos** sem interrupções

---

## Passo 1 — Adicionar o marketplace (1 min)

No Claude Cowork:

1. Abra **Plugins → Marketplace → Adicionar**
2. Cole esta URL:
   ```
   https://github.com/sbroggioadv/triagem-pro-marketplace
   ```
3. Confirme

---

## Passo 2 — Instalar o plugin (30 segundos)

1. Na lista de plugins, localize **"Triagem pro"**
2. Clique no **`+`** (Install)
3. Confirme — o Cowork baixa do GitHub automaticamente

Pronto. O plugin aparece na sidebar em **Habilidades**.

---

## Passo 3 — Configurar (15 min)

No chat do Cowork, digite:

```
/configurar
```

O agente vai te guiar **conversando**, uma pergunta por vez, por 5 etapas:

1. **Verificar Python** (geralmente automático no Cowork)
2. **Conectar WhatsApp via Zappfy** — você cria conta gratuita em [dash.zappfy.io](https://dash.zappfy.io), escaneia QR Code no celular, e cola o token no chat
3. **Configurar escritório** — nome, sua identificação no WhatsApp, equipe (se houver), thresholds de atraso
4. **Aprender sua voz** — agente lê suas últimas mensagens enviadas e gera perfil de estilo
5. **Primeira triagem demonstrativa** — você vê o produto funcionando

---

## Pronto. Como usar a partir de agora

| Você diz / digita | O agente faz |
|---|---|
| `verifica meu zap` | Tabela de pendências por urgência |
| `redige resposta pra [cliente]` | Devolve 1-3 opções no seu estilo + OAB |
| `manda a 1` (após ver opções) | Envia pela Zappfy |
| `/sem-resposta` | Pendências mais antigas |
| `/resumo-dia` | Snapshot 24h |
| `/aprender-voz` | Re-aprende seu estilo |

---

## Segurança em 1 frase

Tudo roda **na sua sessão Cowork**. Nenhum dado seu passa por servidor nosso. Nenhuma telemetria. Sigilo profissional inviolável.
